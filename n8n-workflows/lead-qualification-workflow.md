# Lead Qualification Workflow

## Objective
Practice automation logic in n8n using IF conditions and branching.

## Workflow Structure

Trigger
↓
Create Lead Data
↓
Check Lead Score
↓
Qualified or Rejected

## Example Lead

Name: John Smith
Company: TechFlow
Score: 85

## Logic

If score > 80:
- Lead becomes Qualified

Else:
- Lead becomes Rejected

## What I Learned

- How triggers work
- How nodes connect
- How IF conditions work
- How workflow branching works
- How automation makes decisions
